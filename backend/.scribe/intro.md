# Introduction

Complete API documentation for Shaka Point of Sales system. Includes authentication, user management, product management, and transaction processing.

<aside>
    <strong>Base URL</strong>: <code>http://localhost:8000</code>
</aside>

    Welcome to the Shaka POS API documentation. This API powers our Point of Sales system built with Laravel 12+ and provides endpoints for:
    
    - **Authentication & Authorization** - User login, logout, and role-based access control
    - **User Management** - User profiles and role management  
    - **Product Management** - Products, categories, and inventory (Coming in Sprint 2)
    - **Transaction Processing** - POS transactions and payments (Coming in Sprint 3)
    - **Reporting** - Sales reports and analytics (Coming in Sprint 5)

    <aside>The API uses Laravel Sanctum for authentication. Include your API token in the Authorization header for all authenticated requests.</aside>
